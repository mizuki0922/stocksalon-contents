## 2. 臨時市況（重大局面）

### 基本情報

| 項目 | 内容 |
|------|------|
| 頻度 | **不定期**（重大局面のみ） |
| 配信タイミング | 相場急変時など |
| 配信場所 | Discord #市況まとめ |
| 担当者 | 講座責任者 |

### 発動条件（確定）

以下の**いずれか**に該当した場合に配信：

1. 日経平均が1日で **±3%以上** 変動
2. 日銀の金融政策決定会合で **サプライズ発表**
3. 米国市場（S&P500等）が1日で **±3%以上** 変動
4. 為替（ドル円）が1日で **±2円以上** 変動
5. その他、講座責任者が **会員に伝えるべき**と判断した場合

### 決定事項

- [x] 臨時配信の基準を確定

## 3. メルマガ

### 基本情報

| 項目 | 内容 |
|------|------|
| 頻度 | **3日に1回** |
| 配信時間 | **朝 7:00** |
| 配信場所 | メール |
| 担当者 | 講座責任者 or 運営 |

### 配信フロー

```
【事前】
メルマガ原稿を作成（30日分ストック推奨）
    ↓
【配信設定】
メール配信サービスでステップメール設定
    ↓
【配信日】
自動配信
```

### 決定事項

- [x] 配信時間を確定：**朝 7:00**
- [ ] メルマガのストック本数を確定
- [ ] 使用するメール配信サービスを確定（技術担当と相談後）

### メルマガ管理

| 項目 | 内容 |
|------|------|
| 現在のストック | Day1〜Day30（※要確認） |
| 目標ストック | 常に30日分以上 |
| 原稿作成担当 | 講座責任者 or ライター |

【開催1週間前】
