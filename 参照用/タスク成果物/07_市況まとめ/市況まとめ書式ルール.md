# 市況まとめ 書式ルール

> **目的**: 投稿の品質を一定に保ち、法的リスクを避ける

---

## 基本ルール

| 項目 | ルール |
|------|--------|
| 文字数 | **500〜1,000文字**程度（読みやすい長さ） |
| 配信頻度 | 週2回（火曜・金曜 20:00） |
| 投稿場所 | Discord #市況まとめ |
| 形式 | テンプレートに沿って作成 |

---

## 必須項目

毎回必ず含める内容です。

| # | 項目 | 内容 |
|---|------|------|
| 1 | タイトル | 「市況まとめ（〇月〇日）」 |
| 2 | 相場概況 | 日経平均・TOPIX・ドル円の数値 |
| 3 | 注目ポイント | 3つ程度 |
| 4 | コメント | あい先生の所感 |
| 5 | 次回配信日 | 次回の日付と時間 |

---

## 数字の書き方

| 項目 | 書き方 | 例 |
|------|--------|-----|
| 日経平均 | カンマ区切り + 「円」 | 38,500円 |
| TOPIX | カンマ区切り（円なし） | 2,650 |
| ドル円 | 小数点1桁 + 「円」 | 150.5円 |
| パーセント | 小数点1桁 + 「%」 | +1.2% |
| 前週比 | 「前週比」を明記 | 前週比 +1.2% |
| 前日比 | 「前日比」を明記（臨時配信時） | 前日比 -3.5% |

---

## 禁止表現

以下の表現は**絶対に使わない**でください。金融商品取引法に抵触する恐れがあります。

### 断定表現（NG）

| ❌ 使ってはいけない | ⭕ 言い換え例 |
|---------------------|---------------|
| 必ず上がる | 上昇が期待される |
| 絶対に儲かる | 利益が出る可能性がある |
| 確実に〇〇 | 〇〇の可能性が高い |
| 間違いなく | 〇〇と考えられる |

### 売買推奨（NG）

| ❌ 使ってはいけない | ⭕ 言い換え例 |
|---------------------|---------------|
| 今すぐ買うべき | 注目したい銘柄として |
| この銘柄を買ってください | 〇〇セクターに注目しています |
| 売った方がいい | 慎重な判断が必要かもしれません |
| 〇〇株がおすすめ | 〇〇セクターが堅調です |

### その他の禁止事項

- 特定の個人・企業への誹謗中傷
- 未公開情報（インサイダー情報）の記載
- 他サービス・他者の投資情報の無断転載
- 政治的・宗教的な主張

## 推奨表現

### 見通しを述べるとき

```
・〇〇と考えられます
・〇〇の可能性があります
・〇〇に注目しています
・〇〇かもしれません
・〇〇を意識した展開になりそうです
```

### 注意を促すとき

```
・慎重な判断が必要です
・リスクにも注意が必要です
・急な変動に備えておきましょう
・投資判断は自己責任でお願いします
```

---

## 改行・見出しルール

| ルール | 例 |
|--------|-----|
| 見出しの前は1行空ける | ▼ 注目ポイント（前に空行） |
| 箇条書きは「・」または「1. 2. 3.」 | ・日経平均：38,500円 |
| 長い文は2〜3文で改行 | 〇〇です。△△しました。 |
| 区切り線は「━」を使用 | ━━━━━━━━━━ |

---

## 絵文字の使用

| 使用場所 | 推奨絵文字 |
|----------|------------|
| タイトル | 📊 |
| 臨時配信 | 🚨 |
| 見出しには使わない | ▼ のみ |

絵文字の多用は避け、シンプルに保ちます。
