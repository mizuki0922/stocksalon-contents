# 市況まとめ 配信手順書

> **目的**: 担当者が変わっても、同じ手順で市況まとめを配信できるようにする

---

## 基本情報

| 項目 | 内容 |
|------|------|
| 配信頻度 | **週2回**（火曜・金曜） |
| 配信時間 | **20:00** |
| 配信場所 | Discord **#市況まとめ** チャンネル |
| 原稿作成 | 講座責任者 |
| 配信担当 | 運営スタッフ（小谷） |

---

## 配信の流れ（全体図）

```
【配信日 当日】

  講座責任者が原稿を作成
        ↓
  18:00までに原稿を受領
        ↓
  内容を確認（誤字脱字・禁止表現）
        ↓
  問題なければ「OK」と返信
        ↓
  20:00にDiscordへ投稿
        ↓
  投稿完了を講座責任者に報告
```

---

## 手順詳細

### STEP 1: 原稿を受け取る

| 項目 | 内容 |
|------|------|
| 締切 | 配信日当日 **18:00** |
| 受け取り方法 | （チャット or 共有フォルダ）※要確認 |
| 届かない場合 | 17:30までに届かなければ講座責任者に確認連絡 |

**確認連絡の例文:**
```
お疲れ様です。本日の市況まとめの原稿ですが、
まだ届いていないようです。
18:00締切で20:00配信予定ですが、
本日の配信は可能でしょうか？
```

---

### STEP 2: 原稿を確認する

以下の項目をチェックします。

**チェックリスト:**

| # | チェック項目 | 確認 |
|---|-------------|------|
| 1 | 誤字・脱字はないか | ⬜ |
| 2 | 日付は正しいか | ⬜ |
| 3 | 数字（日経平均、ドル円など）は正しいか | ⬜ |
| 4 | 禁止表現が含まれていないか | ⬜ |
| 5 | テンプレートの形式に沿っているか | ⬜ |

**禁止表現（使ってはいけない言葉）:**
- 「必ず儲かる」「絶対に上がる」などの断定表現
- 特定銘柄の売買推奨（「今すぐ買うべき」など）
- 投資勧誘にあたる表現

**問題があった場合:**
- 講座責任者に修正箇所を伝える
- 修正版を受け取って再確認

---

### STEP 3: OKを返す

確認が完了したら、講座責任者に「OK」を伝えます。

**OK連絡の例文:**
```
原稿確認しました。問題ありません！
20:00に配信します。
```

---

### STEP 4: Discordに投稿する

**投稿手順:**

1. Discordを開く
2. サーバー「**あい先生｜株コンパスサロン運営**」を選択
3. 左側のチャンネル一覧から **#市況まとめ** をクリック
4. 画面下部の入力欄に原稿を貼り付け
5. 内容を最終確認
6. **Enter** キーで送信（または送信ボタンをクリック）

**投稿時間:** 20:00 ぴったり（±5分程度はOK）

---

### STEP 5: 投稿完了を報告する

投稿が完了したら、講座責任者に報告します。

**報告の例文:**
```
市況まとめ、配信完了しました！
```

## 臨時配信の場合

通常の火曜・金曜以外に、講座責任者から「臨時で配信したい」と連絡があった場合。

### 臨時配信の発動条件

1. 日経平均が1日で ±3%以上 変動
2. 日銀の金融政策決定会合でサプライズ発表
3. 米国市場（S&P500等）が1日で ±3%以上 変動
4. 為替（ドル円）が1日で ±2円以上 変動
5. 講座責任者が会員に伝えるべきと判断した場合

### 臨時配信の手順

1. 講座責任者から連絡を受ける
2. 原稿を受け取る（通常より短い締切でOK）
3. 確認 → OK → 投稿
4. **#運営からのお知らせ** と **LINE** で臨時配信をお知らせ

### 臨時配信のお知らせテンプレート

**Discord #運営からのお知らせ:**
```
📢 臨時市況まとめを配信しました！
本日は相場が大きく動いたため、臨時で配信しています。
#市況まとめ チャンネルをご確認ください。
```

**LINE:**
```
📢 臨時市況まとめを配信しました！
本日は相場が大きく動いたため、臨時で配信しています。
Discordの #市況まとめ をご確認ください。
→ https://discord.gg/ez8rbXU6
```

## トラブル対応

### 原稿が届かない場合

1. 17:30までに届かなければ講座責任者に確認
2. 連絡がつかない場合 → 配信をスキップ
3. 配信予定表に「スキップ」と理由を記載

### 投稿を間違えた場合

1. 間違えた投稿を削除（メッセージを右クリック→削除）
2. 正しい内容で再投稿
3. 大きな間違いだった場合は、お詫びを添える

**お詫びの例文:**
```
先ほどの投稿に誤りがありましたので、修正して再投稿いたしました。
失礼いたしました。
```
