# 05_Discord

コミュニティは BAND ではなく **Discord** で運用。Bot による会員ロール付与・剥奪でアクセス制御。

## このフォルダに格納する成果物

- Discord サーバー・チャンネル構成の設計
- Discord Bot の設定メモ（ロール付与・剥奪、会員サイトとの連携）
- 会員サイトからの Discord 連携フロー（認証コード方式でのユーザー紐づけ）
- ルール掲示・権限設定の設計
- Discord 運営マニュアル（管理者向け・会員向け）
- Discord 利用規約・チャンネルルール
- コミュニティ活性化・チャンネル最適化のメモ

作業のたびに、このフォルダにファイルを1つずつ追加していく。
