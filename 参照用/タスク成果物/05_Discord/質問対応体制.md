# Discord 質問対応体制

## 概要

Discord内で会員から質問が来た際の対応フローを定義します。
「受け取る → 振る → 返信 → 記録」の流れで運用します。

---

## 対応ルール早見表（まず確認）

質問が来たら、まずこの表を確認してください。

### ✅ 運営スタッフが答えてOK

| カテゴリ | 具体例 |
|---------|--------|
| **Discord操作** | 使い方、通知設定、メッセージの送り方、招待リンク |
| **ログイン関連** | 会員サイトにログインできない、パスワードリセット |
| **サービス内容** | 配信日程（市況まとめ、勉強会）、入会特典の内容 |
| **FAQ記載事項** | このドキュメントのFAQに載っている内容 |

---

### ❌ 答えてはいけない（絶対NG）

| カテゴリ | 具体例 |
|---------|--------|
| **個別銘柄の推奨** | 「〇〇株は買いですか？」「おすすめ銘柄を教えて」 |
| **投資の判断** | 「今売るべきですか？」「いくらで買えばいい？」 |
| **法律・税務** | 「確定申告はどうすれば？」「この取引は違法？」 |
| **医療・健康** | 「投資のストレスで体調が…」 |

→ これらは **回答せず、エスカレーション** してください。

---

### 📤 エスカレーション先（誰に渡すか）

| 質問の内容 | 連絡先 | 連絡方法 |
|-----------|--------|---------|
| 投資判断・相場見解・銘柄分析 | **講座責任者** | Slack / LINE / Discord DM |
| システムトラブル（決済エラー、サイト障害） | **技術担当** | Slack / メール |
| クレーム・トラブル・退会希望 | **ディレクター** | Slack / メール |
| 判断に迷う質問 | **ディレクター** | Slack |

---

### ⚠️ 迷ったときの判断基準

- **FAQに載っている** → 自分で回答OK
- **FAQに載っていないが、事実を伝えるだけ** → 自分で回答OK
- **意見や判断を求められている** → エスカレーション
- **よくわからない** → エスカレーション（無理に答えない）

---

## ① 受け取る（確認）

### 当番制 vs 全員確認

| 方式 | メリット | デメリット | 推奨場面 |
|------|---------|-----------|---------|
| **当番制** | 責任が明確、見落とし防止 | 当番不在時に遅延 | スタッフ2名以上 |
| **全員確認** | 柔軟に対応可能 | 「誰かがやるだろう」で放置リスク | スタッフ1名 or 少人数 |

### 推奨：当番制（週替わり）

| 曜日 | 当番 |
|------|------|
| 月〜水 | スタッフA |
| 木〜日 | スタッフB |

※スタッフが1名の場合は「毎日確認」でOK

### 確認のルール

1. **確認タイミング**：1日2回以上（朝・夜など）
2. **確認の証拠**：質問投稿に 👀 リアクションをつける
3. **目標時間**：質問投稿から **24時間以内** に一次回答

### 確認時のアクション

```
質問を発見
    ↓
👀 リアクションをつける（「見ました」の意味）
    ↓
内容を確認し、次のステップへ
```

## ② 振る（判断）

### 振り分け基準

| 質問の種類 | 対応者 | 例 |
|-----------|--------|-----|
| **操作系** | 運営スタッフ | Discord使い方、ログイン方法、通知設定 |
| **サービス系** | 運営スタッフ | 入会特典、勉強会の日程、配信予定 |
| **投資系（基本）** | 運営 or 講座責任者 | 用語の意味、過去配信の補足 |
| **投資系（専門）** | 講座責任者 | 相場見解、売買判断、銘柄分析 |

### 判断フローチャート

```
質問内容を確認
    │
    ├─ Discordの操作・使い方？
    │       → 運営が回答（FAQを参照）
    │
    ├─ サービスについて？（日程、特典など）
    │       → 運営が回答
    │
    ├─ 投資の基本的な質問？（用語、仕組みなど）
    │       → 運営が回答可能なら回答
    │       → 迷ったら講座責任者に確認
    │
    └─ 投資判断・相場見解・銘柄について？
            → 講座責任者に連携
```

### 講座責任者への連携方法

| 連絡手段 | 用途 |
|---------|------|
| Slack/LINE | 「#質問・相談 に投資判断の質問が来ています。ご確認お願いします」 |
| Discord DM | 同上（講座責任者がDiscordを見ている場合） |

**連絡テンプレート**
```
【質問対応依頼】
・チャンネル：#質問・相談
・投稿者：〇〇さん
・質問内容：〇〇〇〇（要約）
・URL：（該当メッセージへのリンク）

ご確認お願いいたします。
```

## ③ 返信（回答）

### 回答の基本ルール

| 項目 | ルール |
|------|--------|
| **返信速度** | 24時間以内（目標）、遅くとも48時間以内 |
| **トーン** | 丁寧・親しみやすく |
| **敬称** | 「〇〇さん」と呼ぶ |
| **投資判断** | 「〜だと思います」は避け、一般論で回答 |

### 回答テンプレート

**一般的な回答**
```
〇〇さん、ご質問ありがとうございます！

（回答内容）

ご不明点があれば、お気軽にご質問くださいね。
```

**すぐに回答できない場合**
```
〇〇さん、ご質問ありがとうございます！

確認してからお答えしますので、少々お待ちくださいね。
```

**講座責任者から回答する場合**
```
〇〇さん、ご質問ありがとうございます！

（講座責任者名）からお答えします。

（回答内容）
```

### 回答時の注意点

- ❌ 「絶対に上がる」「買うべき」など断定的な投資助言はNG
- ❌ 特定銘柄の売買推奨はNG
- ✅ 一般的な考え方・判断基準の説明はOK
- ✅ 過去の配信内容への補足はOK

---

## ④ 記録（任意）

初期は記録しなくてもOKです。以下の場合に記録を始めることを検討してください。

- 同じ質問が3回以上来た
- スタッフが増えて引き継ぎが必要になった
- 講座責任者の専門的な回答を残したい

### 記録方法：#対応ログ チャンネルに投稿

Discord内に **#対応ログ**（運営専用チャンネル）を作成し、対応後に簡単にメモを投稿します。

**投稿フォーマット**
```
【対応ログ】3/1
質問：通知が多すぎる
回答：通知設定の変更方法を案内
対応：スタッフA
```

※細かく書く必要はありません。後から見て「こんな質問があったな」とわかればOK

## 対応体制まとめ

### 役割分担

| 役割 | 担当者 | 主な業務 |
|------|--------|---------|
| **一次対応** | 運営スタッフ | 質問確認、振り分け、操作系回答 |
| **専門回答** | 講座責任者 | 投資判断・相場見解の回答 |

### 対応時間の目安

| 質問の種類 | 目標時間 |
|-----------|---------|
| 操作・サービス系 | 24時間以内 |
| 投資基礎 | 24〜48時間以内 |
| 投資専門（講座責任者対応） | 48時間以内 |

### 不在時の対応

| 状況 | 対応 |
|------|------|
| 当番スタッフが休み | もう一人のスタッフがカバー |
| 全スタッフ不在 | 「確認中です」のリアクション（⏳）をつけておく |
| 講座責任者が多忙 | 「後日回答します」と一次返信 |

## FAQ・コピペ用返信文集

`〇〇さん` の部分を質問者の名前に置き換えてください。

### 操作系（Discord）

#### Q. 通知が多すぎます

```
〇〇さん、ご質問ありがとうございます！

通知が多い場合は、以下の手順で調整できます。

1. サーバー名（株コンパスサロン）を右クリック
2. 「通知設定」をクリック
3. 「@メンションのみ」を選択

これで必要な通知だけ届くようになります。
他にご不明点があれば、お気軽にお知らせくださいね！
```

---

#### Q. 過去のメッセージを見たい

```
〇〇さん、ご質問ありがとうございます！

過去のメッセージは、チャンネル内で上にスクロールすると読めます。

また、画面上部の検索ボックス（虫眼鏡アイコン）を使うと、キーワードで検索することもできます。

他にご不明点があれば、お気軽にどうぞ！
```

---

#### Q. メッセージが送れません

```
〇〇さん、ご質問ありがとうございます！

#運営からのお知らせ や #市況まとめ などのチャンネルは「読み取り専用」のため、書き込みができない設定になっています。

メッセージを送りたい場合は、#雑談 や #質問・相談 チャンネルをお試しください！

他にご不明点があれば、お気軽にどうぞ！
```

---

#### Q. Discordに参加できません / 招待リンクが開けません

```
〇〇さん、ご質問ありがとうございます！

招待リンクはこちらです。
→ https://discord.gg/ez8rbXU6

リンクが開けない場合は、以下の方法をお試しください。

1. Discordアプリを開く
2. 左下の「＋」をクリック
3. 「サーバーに参加」を選択
4. 上記のリンクを貼り付けて「参加」をクリック

それでも参加できない場合は、お知らせくださいね！
```

### サービス系（入会特典・コンテンツ）

#### Q. 入会特典は何がありますか？

```
〇〇さん、ご質問ありがとうございます！

入会特典として、以下の3つのコンテンツをご用意しています。
この順番で見ると、投資の基礎が身につきます。

1. 投資のスタイル・銘柄の選び方
2. いつ買うのか（買いのタイミング）
3. いつ売るのか（売りのタイミング）

#運営からのお知らせ チャンネルに詳しい案内とリンクがありますので、ぜひご覧ください！

他にご不明点があれば、お気軽にどうぞ！
```

---

#### Q. 特典はどこで受け取れますか？

```
〇〇さん、ご質問ありがとうございます！

特典コンテンツは、会員サイトからダウンロードできます。

1. 会員サイトにログイン
2. 特典コンテンツのページへ移動
3. ダウンロードボタンをクリック

ログイン情報は、入会完了メールに記載されています。
他にご不明点があれば、お気軽にどうぞ！
```

---

#### Q. 特典の動画が見れません

```
〇〇さん、ご連絡ありがとうございます！

動画が見れない場合、以下をお試しください。

1. 会員サイトにログインしているかご確認ください
2. ログインできない場合は、入会完了メールに記載のID・パスワードをお試しください
3. パスワードを忘れた場合は、ログイン画面の「パスワードを忘れた方」からリセットできます

それでも解決しない場合は、お知らせくださいね！
```
