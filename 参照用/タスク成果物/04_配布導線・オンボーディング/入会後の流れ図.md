# 入会後の流れ図

## 概要

会員が入会してから特典を受け取るまでの流れを図にしたものです。
技術担当に確認してもらい、画面の動きや仕様と合っているかチェックしてください。

---

## 入会後の流れ（全体図）

```
┌─────────────────────────────────────────────────────────────────┐
│                        【入会の流れ】                            │
└─────────────────────────────────────────────────────────────────┘

    ┌───────────────┐
    │  ① 決済完了   │
    │  （Stripe等）  │
    └───────┬───────┘
            │
            ▼
    ┌───────────────┐     届くもの：入会完了メール
    │ ② メール受信  │     内容：会員サイトURL、ログイン情報
    │               │
    └───────┬───────┘
            │
            ▼
    ┌───────────────┐     やること：メール内のURLをクリック
    │ ③ 会員サイト  │              ログイン（ID/パスワード入力）
    │   ログイン    │
    └───────┬───────┘
            │
            ▼
    ┌───────────────┐     やること：会員サイト内のリンクをタップ
    │ ④ LINE登録   │              LINE友だち追加
    │  （友だち追加）│
    └───────┬───────┘
            │
            ▼
    ┌───────────────┐     やること：招待リンクをクリック
    │ ⑤ Discord参加 │              アカウント作成（初めての場合）
    │               │              サーバーに参加
    └───────┬───────┘
            │
            ▼
    ┌───────────────┐     届くもの：
    │ ⑥ 特典受け取り │     ・銘柄選定チェックリスト
    │               │     ・判断軸コンテンツ3本（PDF）
    └───────┬───────┘     ・メルマガ（3日に1回）
            │
            ▼
    ┌───────────────┐
    │ ⑦ 利用開始！  │
    │               │
    └───────────────┘
```

---

## 各ステップの詳細

### ① 決済完了

| 項目 | 内容 |
|------|------|
| 会員がやること | 決済ページで支払い |
| 使用するシステム | Stripe（または他の決済サービス） |
| 完了の目印 | 「決済完了」画面が表示される |

---

### ② メール受信

| 項目 | 内容 |
|------|------|
| 届くもの | 入会完了メール |
| 届くタイミング | 決済完了後すぐ（自動送信） |
| メールに含まれる内容 | ・会員サイトのURL<br>・ログインID（メールアドレス）<br>・初期パスワード or パスワード設定リンク<br>・問い合わせ先 |
| 送信元 | aisensei.stock@gmail.com（または配信サービス） |

**▶ 技術確認ポイント**
- [ ] 決済完了後、自動でメールが送信されるか？
- [ ] メールの送信元アドレスは何か？
- [ ] ログイン情報はどのように発行されるか？

---

### ③ 会員サイトログイン

| 項目 | 内容 |
|------|------|
| 会員がやること | メール内のURLをクリック → ログイン画面でID/パスワード入力 |
| 会員サイトURL | https://[会員サイトURL]（※要確認） |
| ログイン方法 | メールアドレス + パスワード |

**▶ 技術確認ポイント**
- [ ] 会員サイトのURLは何か？
- [ ] ログイン方法は？（メール+パスワード？ソーシャルログイン？）
- [ ] パスワードは自動発行？会員が設定？

---

### ④ LINE登録

| 項目 | 内容 |
|------|------|
| 会員がやること | 会員サイト内のLINEリンクをタップ → 友だち追加 |
| LINE公式アカウントURL | https://line.me/R/ti/p/[アカウントID]（※要確認） |
| 登録後に届くもの | LINEウェルカムメッセージ |

**▶ 技術確認ポイント**
- [ ] LINE公式アカウントのURLは何か？
- [ ] 会員サイトのどこにLINEリンクがあるか？
- [ ] LINE登録後の自動メッセージは設定済みか？

---

### ⑤ Discord参加

| 項目 | 内容 |
|------|------|
| 会員がやること | 招待リンクをクリック → アカウント作成（初めての場合） → サーバー参加 |
| 招待リンク | https://discord.gg/ez8rbXU6 |
| リンクの掲載場所 | ・入会完了メール<br>・会員サイト<br>・LINE |

**▶ 技術確認ポイント**
- [ ] 招待リンクはどこに掲載するか？（メール/会員サイト/LINE）
- [ ] 招待リンクの有効期限は無期限になっているか？

---

### ⑥ 特典受け取り

| 特典 | 受け取り場所 | 受け取り方法 |
|------|-------------|-------------|
| 銘柄選定チェックリスト | 会員サイト | ダウンロード |
| 判断軸コンテンツ3本（PDF） | 会員サイト | ダウンロード |
| メルマガ | メール | 自動配信（3日に1回） |
| 基礎テーマ録画 | 会員サイト | 視聴 |

**▶ 技術確認ポイント**
- [ ] 特典ファイルは会員サイトのどこに置くか？
- [ ] メルマガの自動配信は設定済みか？
- [ ] 基礎テーマ録画のアップロード先は？

---

### ⑦ 利用開始

| サービス | 届く場所 | 頻度 |
|---------|---------|------|
| 市況まとめ | Discord #市況まとめ | 週2回 |
| メルマガ | メール | 3日に1回 |
| オンライン勉強会 | Zoom（Discordで告知） | 月1回 |
| 質問対応 | Discord #質問・相談 / LINE | 随時 |

## 会員から見た画面遷移（簡易版）

```
【スマホの場合】

決済ページ（支払い）
    ↓ 「購入する」ボタン
決済完了画面
    ↓
メールアプリ（入会完了メールを開く）
    ↓ URLをタップ
会員サイト（ログイン画面）
    ↓ ID/パスワード入力
会員サイト（トップページ）
    ↓ 「LINE登録」ボタン
LINEアプリ（友だち追加画面）
    ↓ 「追加」ボタン
LINE（ウェルカムメッセージ受信）
    ↓
会員サイトに戻る
    ↓ 「Discord参加」ボタン
Discordアプリ（サーバー参加画面）
    ↓ 「参加する」ボタン
Discord（株コンパスサロン）
    ↓
会員サイトに戻る
    ↓ 「特典ダウンロード」ボタン
特典PDF表示
```

## 技術担当への確認依頼

以下の点を技術担当に確認してください。

### 確認事項チェックリスト

**決済・メール関連**
- [ ] 決済完了後、自動でメールが送信される仕組みになっているか
- [ ] メールの送信元アドレスは何か
- [ ] ログイン情報（ID/パスワード）の発行方法

**会員サイト関連**
- [ ] 会員サイトのURL
- [ ] ログイン方法（メール+パスワード or その他）
- [ ] 会員サイトにLINE登録リンクを設置できるか
- [ ] 会員サイトにDiscord招待リンクを設置できるか
- [ ] 特典ファイルのアップロード場所

**LINE関連**
- [ ] LINE公式アカウントのURL
- [ ] LINE登録後の自動メッセージ設定

**その他**
- [ ] 上記フローで技術的に問題がある箇所はないか
- [ ] 追加で必要な設定はないか
