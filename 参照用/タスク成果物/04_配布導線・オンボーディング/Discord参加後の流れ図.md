# Discord参加後の流れ図

> **目的**: 入会してDiscordに参加した会員が、最初に何をすればよいか迷わないようにする

---

## フロー図（テキスト版）

```
┌─────────────────────────────────────────────────────────────────┐
│                    Discord参加後の流れ                           │
└─────────────────────────────────────────────────────────────────┘

    ┌──────────────┐
    │  Discord参加  │  ← 招待リンクをクリックしてサーバーに入る
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐
    │  ルール確認   │  ← #ルール チャンネルを読む
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐
    │  自己紹介     │  ← #自己紹介 チャンネルで挨拶（任意）
    │   （任意）    │
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐
    │  操作確認     │  ← #操作方法・よくある質問 を確認
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐
    │  特典確認     │  ← #運営からのお知らせ で特典コンテンツを確認
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐
    │  利用開始！   │  ← 各チャンネルを自由にご覧ください
    └──────────────┘
```

---

## 各ステップの詳細

### STEP 1: Discord参加
| 項目 | 内容 |
|------|------|
| **何をする？** | 招待リンクをクリックしてサーバーに参加する |
| **どこを見る？** | 入会完了メールまたはLINEに記載の招待リンク |
| **補足** | Discordアカウントがない方は先にアカウント作成が必要 |

### STEP 2: ルール確認
| 項目 | 内容 |
|------|------|
| **何をする？** | サロンのルールを読んで理解する |
| **どこを見る？** | **#ルール** チャンネル |
| **所要時間** | 約2〜3分 |
| **補足** | 誹謗中傷禁止、投資勧誘禁止などの基本ルールを確認 |

### STEP 3: 自己紹介（任意）
| 項目 | 内容 |
|------|------|
| **何をする？** | 簡単な自己紹介を投稿する |
| **どこを見る？** | **#自己紹介** チャンネル |
| **所要時間** | 約1〜2分 |
| **補足** | 投稿は任意ですが、他の会員との交流のきっかけになります |
| **書く内容の例** | ニックネーム、投資歴、興味のあること など |

### STEP 4: 操作方法の確認
| 項目 | 内容 |
|------|------|
| **何をする？** | Discordの基本操作とよくある質問を確認する |
| **どこを見る？** | **#操作方法・よくある質問** チャンネル |
| **所要時間** | 約3〜5分 |
| **補足** | チャンネルの見方、通知設定、質問の仕方などを確認 |

### STEP 5: 特典コンテンツの確認
| 項目 | 内容 |
|------|------|
| **何をする？** | 入会特典のコンテンツを確認する |
| **どこを見る？** | **#運営からのお知らせ** チャンネル |
| **補足** | 最初に見てほしい動画や資料の案内があります |

### STEP 6: 利用開始！
| 項目 | 内容 |
|------|------|
| **何をする？** | 各チャンネルを自由に利用する |
| **主なチャンネル** | #市況まとめ、#質問、#雑談 など |
| **補足** | 分からないことがあれば#質問チャンネルへ |

## 会員向け案内文（Discord投稿用）

以下の文章を **#運営からのお知らせ** または **ウェルカムメッセージ** として使用できます。

---

### 📌 ご入会ありがとうございます！最初にやることガイド

サロンへようこそ！  
以下の順番で進めていただくと、スムーズにご利用いただけます。

**① #ルール を読む（2〜3分）**  
サロンの基本ルールをご確認ください。

**② #自己紹介 で挨拶する（任意）**  
よろしければ簡単な自己紹介をどうぞ。他の会員との交流のきっかけになります。

**③ #操作方法・よくある質問 を確認する（3〜5分）**  
Discordの使い方や、よくある質問をまとめています。

**④ 特典コンテンツを見る**  
入会特典として、以下のコンテンツをご用意しています。  
（※ここに特典コンテンツのリンクを記載）

**⑤ 各チャンネルを自由にご利用ください**  
- #市況まとめ：週2回の市況レポート  
- #質問：分からないことはこちらへ  
- #雑談：会員同士の交流にどうぞ  

ご不明な点があれば、お気軽に #質問 チャンネルでお聞きください！
