# オンボーディング設計

## 概要

新規会員が入会後、スムーズにサロンを活用できるようにするための設計です。
「何をすればいいかわからない」状態をなくし、早期に価値を感じてもらうことが目的です。

---

## オンボーディングの目標

| 期間 | 目標 |
|------|------|
| 入会直後 | Discord参加・特典受け取り完了 |
| 1週間以内 | 基礎コンテンツを1つ以上視聴/読了 |
| 2週間以内 | 市況まとめを2回以上受け取る |
| 1ヶ月以内 | 勉強会に1回参加 or 録画視聴 |

---

## オンボーディング全体フロー

```
【Day 0】入会・決済完了
    ↓
【Day 0】入会完了メール受信
    │  ├─ Discord招待リンク
    │  ├─ 特典の受け取り方法
    │  └─ 最初にやることリスト
    ↓
【Day 0〜1】Discord参加
    │  ├─ #ルール を確認
    │  ├─ #自己紹介 で挨拶（任意）
    │  └─ #操作方法・よくある質問 を確認
    ↓
【Day 1】ウェルカムメッセージ（自動 or 手動）
    │  └─ 特典コンテンツの案内
    ↓
【Day 1〜3】特典を受け取る
    │  ├─ 銘柄選定チェックリスト
    │  ├─ 判断軸コンテンツ3本（PDF）
    │  └─ メルマガ登録確認
    ↓
【Day 3】フォローアップメール①
    │  └─ 基礎テーマ録画の視聴案内
    ↓
【Day 7】フォローアップメール②
    │  └─ 次回勉強会の案内・質問の仕方
    ↓
【Day 14〜】通常運用へ
    ├─ 週2回の市況まとめ
    ├─ 3日に1回のメルマガ
    └─ 月1回の勉強会
```

---

## Day 0：入会完了メール

### 含める内容

1. **お礼と歓迎メッセージ**
2. **Discord招待リンク**
3. **特典の受け取り方法**
   - 銘柄選定チェックリスト
   - 判断軸コンテンツ3本（PDF）
4. **最初にやることリスト**
5. **問い合わせ先**

**テンプレート:** `16_入会フロー/入会完了メール_テンプレート.md`

## 特典配布の導線

| 特典 | 配布方法 | 配布タイミング |
|------|---------|---------------|
| 銘柄選定チェックリスト | 会員サイト or メール添付 | 入会完了メール |
| 判断軸コンテンツ3本（PDF） | 会員サイト or メール添付 | 入会完了メール |
| メルマガ | 自動配信（3日に1回） | 入会翌日から |
| 基礎テーマ録画 | 会員サイト | いつでも視聴可 |

---

## 運営側のチェックリスト

### 事前準備（1回だけ）

- [ ] 入会完了メールのテンプレートを決済システムに設定
- [ ] ウェルカムメッセージをDiscordに準備
- [ ] フォローアップメール①のテンプレートを作成
- [ ] フォローアップメール②のテンプレートを作成
- [ ] 特典ファイルを会員サイトにアップロード
- [ ] 基礎テーマ録画を会員サイトにアップロード
- [ ] メルマガの自動配信を設定

### 新規会員ごと

- [ ] Day 0: 入会完了メールが送信されたことを確認
- [ ] Day 1: Discord参加を確認
- [ ] Day 1: ウェルカムメッセージを送信（自動化できれば不要）
- [ ] Day 3: フォローアップメール①を送信
- [ ] Day 7: フォローアップメール②を送信

---

## 自動化の検討

| 項目 | 自動化方法 |
|------|-----------|
| 入会完了メール | Stripe + メール配信サービス連携 |
| フォローアップメール | メール配信サービスのステップメール機能 |
| ウェルカムメッセージ | Discord Bot（MEE6等） |
| メルマガ | メール配信サービスの定期配信 |
