### 判断軸コンテンツ3本（動画講義）
- **①銘柄選びの判断軸** ②財務指標の見方 ③売買タイミングの判断軸  
- **動画講義**のため、以下が必須：
  - **資料作成**（スライド・台本など）
  - **撮影**
  - **編集**（完成動画）
- 各本ごとの資料・撮影メモ・編集仕様・完成動画の格納先メモなど、作業のたびに1ファイルずつ追加する。
