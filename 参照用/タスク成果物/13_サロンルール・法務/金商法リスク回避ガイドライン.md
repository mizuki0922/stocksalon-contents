# 金商法リスク回避ガイドライン

株コンパスサロンの運営において、金融商品取引法（金商法）およびその他関連法規に抵触しないための運営ガイドラインです。

**このドキュメントの対象者**: 運営スタッフ、コンテンツ制作者、講座責任者、あい先生

---

## 1. 基本方針

### 株コンパスサロンが提供するもの

✅ **判断軸**（考え方・フレームワーク）の提供
✅ **一般的な投資知識**の教育
✅ **市場全体の動向**に関する情報
✅ **コミュニティ**での交流・情報交換の場

### 株コンパスサロンが提供しないもの

❌ **特定銘柄の売買推奨**
❌ **投資判断の代行**
❌ **個別の投資助言**
❌ **売買タイミングの指示**

---

## 2. 金商法の基礎知識

### 投資助言・代理業とは

金融商品取引法第2条第8項第11号に定める「投資助言・代理業」とは、以下のような行為を**業として**行うことです。

> 有価証券の価値等または金融商品の価値等の分析に基づく投資判断に関し、口頭、文書その他の方法により助言を行うことを約し、相手方がそれに対し報酬を支払うことを約する契約を締結し、当該契約に基づき、助言を行うこと

### ポイント

| 要素 | 説明 |
|------|------|
| 投資判断 | 特定の有価証券について、買うべき・売るべき・保有すべきという判断 |
| 助言 | 上記の投資判断を相手に伝えること |
| 報酬 | 助言の対価として金銭を受け取ること |
| 業として | 反復継続して行うこと |

**これらすべてが揃うと、投資助言業の登録が必要**になります。

## 3. 抵触リスクのある表現・行為

### 🚨 絶対に避けるべき表現

以下の表現は、投資助言に該当する可能性が高いため、**絶対に使用しないでください**。

| NG表現 | 問題点 |
|--------|--------|
| 「〇〇（銘柄名）を買ってください」 | 特定銘柄の売買推奨 |
| 「今が〇〇の買い時です」 | 売買タイミングの指示 |
| 「〇〇は絶対上がります」 | 将来の価格を断定 |
| 「〇〇を売った方がいいですよ」 | 個別の投資助言 |
| 「私なら〇〇を買います」 | 売買推奨と解釈される |
| 「〇〇は△△円で利確しましょう」 | 具体的な売買指示 |
| 「〇〇をポートフォリオに入れるべき」 | 投資判断の代行 |

### ⚠️ 注意が必要な表現

以下の表現は、文脈によっては問題になる可能性があります。

| 注意表現 | 対処法 |
|----------|--------|
| 「〇〇は良い銘柄です」 | 「〇〇は〜という点で注目されています」に言い換え |
| 「〇〇がおすすめ」 | 使用しない。「〇〇を例に解説します」に言い換え |
| 「〇〇は割安です」 | 「〇〇のPERは〜です。割安かどうかは各自ご判断ください」 |
| 「〇〇を持っています」 | 事実の報告はOKだが、推奨と取られないよう注意 |

## 4. OK表現・推奨される表現

### ✅ 安全な表現パターン

| OK表現 | 解説 |
|--------|------|
| 「〇〇を**例として**解説します」 | 教育目的であることが明確 |
| 「〇〇の**見方**を説明します」 | 判断軸の提供 |
| 「〇〇かどうかは**ご自身で判断**ください」 | 自己責任の明示 |
| 「〇〇という**考え方**があります」 | 一般論としての情報提供 |
| 「〇〇の**チェックポイント**は〜」 | フレームワークの提供 |
| 「〇〇の**数値を確認**してみましょう」 | 判断材料の提示 |

### ✅ 必ず添える文言

コンテンツの冒頭・末尾に、以下のような文言を入れることを推奨します。

**冒頭用**
```
※本コンテンツは投資判断の参考となる情報提供を目的としており、
特定の金融商品の売買を推奨するものではありません。
```

**末尾用**
```
※投資の最終判断は、ご自身の責任において行ってください。
```

## 5. 場面別ガイドライン

### 5-1. 市況まとめ配信

| 項目 | ガイドライン |
|------|-------------|
| 市場全体の動向 | ✅ OK（「日経平均は〜」「米国市場は〜」） |
| セクター動向 | ✅ OK（「半導体セクターは〜」） |
| 個別銘柄への言及 | ⚠️ 注意（事実の報告のみ、推奨はNG） |
| 今後の見通し | ⚠️ 注意（断定せず「〜の可能性」と表現） |
| 売買推奨 | ❌ NG |

**市況まとめの例文**

```
【OK例】
今週の日経平均は〇〇円で引けました。
半導体セクターは〇〇の影響で上昇しています。
〇〇（銘柄）は決算発表を受けて株価が動きました。

【NG例】
〇〇は今が買い時です。
〇〇はまだ上がるので持ち続けましょう。
```

### 5-2. 勉強会・動画コンテンツ

| 項目 | ガイドライン |
|------|-------------|
| 判断軸の解説 | ✅ OK |
| 財務指標の見方 | ✅ OK |
| 銘柄を例に解説 | ✅ OK（「例として」と明示） |
| 「この銘柄がおすすめ」 | ❌ NG |
| 「私ならこう判断する」 | ⚠️ 注意（「一つの考え方として」と前置き） |

**勉強会での言い回し例**

```
【OK例】
「今日は〇〇を例に、PERの見方を解説します」
「この判断軸で考えると、〇〇はこういう数値になります」
「最終的にどう判断するかは、皆さん自身で決めてくださいね」

【NG例】
「〇〇は絶対買った方がいいです」
「私が〇〇を買った理由は〜だから、皆さんも買いましょう」
```

### 5-3. Discord質問対応

| 項目 | ガイドライン |
|------|-------------|
| 一般的な質問への回答 | ✅ OK |
| 「〇〇を買うべきですか？」への回答 | ❌ 売買推奨はNG。判断軸を伝える |
| 銘柄分析の依頼 | ⚠️ 判断軸に沿った見方を伝える（結論は出さない） |

**質問対応の回答例**

```
【質問】〇〇（銘柄）は買いですか？

【NG回答】
「買いだと思います」「今が買い時です」

【OK回答】
「特定の銘柄についての売買推奨は行っておりません。
〇〇を判断軸で見ると、PERは〜、ROEは〜という数値です。
これらの数値をどう評価するかは、ご自身でご判断くださいね。」
```

### 5-4. メルマガ

| 項目 | ガイドライン |
|------|-------------|
| 判断軸の解説 | ✅ OK |
| 投資マインドの共有 | ✅ OK |
| 「〇〇を買いましょう」 | ❌ NG |
| 個別銘柄の推奨 | ❌ NG |

## 6. その他の法的リスク

### 6-1. 景品表示法

| リスク | 対策 |
|--------|------|
| 「絶対儲かる」「必ず利益が出る」 | 使用禁止。将来の利益を保証する表現はNG |
| 「〇〇万円稼げる」 | 根拠のない数値は使用禁止 |
| 過去の実績の誇大表示 | 「個人の感想です」「成果を保証するものではありません」を付記 |

### 6-2. 特定商取引法

| リスク | 対策 |
|--------|------|
| 事業者情報の非表示 | 特商法に基づく表記を会員サイトに掲載 |
| クーリングオフ | オンラインサービスは対象外だが、説明は必要 |

### 6-3. インサイダー取引規制

| リスク | 対策 |
|--------|------|
| 未公開情報の共有 | 「〇〇社の内部情報によると〜」は絶対NG |
| 会員からの未公開情報 | 共有された場合は即削除、注意喚起 |

### 6-4. 風説の流布・相場操縦

| リスク | 対策 |
|--------|------|
| 根拠のない噂の拡散 | 「〇〇らしい」「〇〇という噂がある」は情報源を確認 |
| 株価操作目的の発言 | 「皆で〇〇を買って株価を上げよう」は絶対NG |

## 8. 問題が発生した場合

### 発見時の対応

1. **該当投稿の削除**（Discord・会員サイト）
2. **投稿者への個別連絡**（会員の場合）
3. **講座責任者への報告**
4. **必要に応じて法務相談**

### 外部からの指摘・問い合わせ

1. **即座に対応せず、講座責任者に報告**
2. **記録を残す**（スクリーンショット等）
3. **必要に応じて法務・弁護士に相談**
4. **回答は慎重に行う**

## 9. よくある質問

### Q1. 「私は〇〇を買いました」と言うのはOK？

**A.** 事実の報告としてはOKですが、「だから皆さんも買いましょう」とつなげるとNGです。

```
OK: 「私は判断軸で検討した結果、〇〇を買いました」
NG: 「私は〇〇を買いました。おすすめです」
```

### Q2. 会員から「〇〇を買うべきか」と聞かれたら？

**A.** 売買の判断は伝えず、判断軸に沿った見方を伝えてください。

```
OK: 「判断軸で見ると〇〇という数値です。どう判断されるかはお任せします」
NG: 「買った方がいいと思います」
```

### Q3. 市況まとめで個別銘柄に触れてはいけない？

**A.** 事実の報告（「〇〇は決算発表で上昇」など）はOKです。「〇〇を買いましょう」はNGです。

### Q4. 「割安だと思います」は言ってもいい？

**A.** 「PERが〇倍なので、割安と判断する人もいます」のように、客観的な数値＋一般論として伝えてください。「割安なので買いましょう」はNGです。

### Q5. 投資助言業の登録は必要？

**A.** 現在のサービス設計（判断軸の提供）であれば、登録は不要と考えられます。ただし、サービス内容を変更する場合は、法務に確認してください。
