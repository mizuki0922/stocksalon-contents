# 12_講座責任者連携

## 役割の前提
- **講座責任者**: コンテンツ制作者へのディレクション（指示・確認・承認）
- **コンテンツ制作者**: 市況まとめ・勉強会資料など原稿・コンテンツの作成
- **あい先生（iSense）**: 演者（勉強会の登壇・出演等）。講座責任者ではない

## このフォルダに格納する成果物

- 業務フローの確立（原稿作成→確認→修正→承認：講座責任者↔コンテンツ制作者）
- 連携ツール・チャットの設定メモ
- 定例ミーティングの設定（月1回）
- 市況まとめ・勉強会資料作成の連携フロー

作業のたびに、このフォルダにファイルを1つずつ追加していく。
