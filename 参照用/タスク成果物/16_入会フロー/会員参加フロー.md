# 会員参加フロー

## 概要

入会から Discord 参加までの流れを定義します。

---

## フロー図

```
┌─────────────────┐
│  1. 入会・決済   │
│  （決済ページ）  │
└────────┬────────┘
         ↓
┌─────────────────┐
│ 2. メール受信    │
│（入会完了メール）│
└────────┬────────┘
         ↓
┌─────────────────┐
│ 3. Discord参加   │
│（招待リンク経由）│
└────────┬────────┘
         ↓
┌─────────────────┐
│ 4. 自己紹介     │
│  （任意）       │
└────────┬────────┘
         ↓
┌─────────────────┐
│ 5. 利用開始！    │
└─────────────────┘
```

---

## 各ステップの詳細

### ステップ1: 入会・決済

| 項目 | 内容 |
|------|------|
| 会員がやること | 決済ページで支払い |
| 運営が用意するもの | 決済ページ（Stripe等） |
| 完了条件 | 決済完了 |

---

### ステップ2: 入会完了メール受信

| 項目 | 内容 |
|------|------|
| 会員がやること | メールを確認 |
| 運営が用意するもの | 入会完了メール（テンプレートあり） |
| メールに含める内容 | Discord招待リンク、サービス説明、問い合わせ先 |
| 完了条件 | メール送信完了 |

**テンプレート:** `入会完了メール_テンプレート.md`

---

### ステップ3: Discord参加

| 項目 | 内容 |
|------|------|
| 会員がやること | 招待リンクをクリック、Discordアカウント作成（未登録の場合） |
| 運営が用意するもの | Discord招待リンク（無期限） |
| 完了条件 | サーバーに参加 |

**招待リンク発行方法:**
1. サーバー名をクリック →「友達を招待」
2. 「招待リンクを編集」→ 有効期限「無期限」、使用回数「無制限」
3. リンクをコピー

**現在の招待リンク:** `https://discord.gg/ez8rbXU6`

---

### ステップ4: 自己紹介（任意）

| 項目 | 内容 |
|------|------|
| 会員がやること | #自己紹介 チャンネルで自己紹介 |
| 運営が用意するもの | 自己紹介テンプレート（チャンネルに固定投稿） |
| 完了条件 | 任意のため完了条件なし |

**テンプレート:** `Discord自己紹介チャンネル_案内文.md`

---

### ステップ5: 利用開始

会員がサロンのサービスを利用開始。

- #市況まとめ で週2回の配信を受け取る
- #質問・相談 やLINEで質問
- 月1回のオンライン勉強会に参加

---

## 運営側のチェックリスト

### 事前準備（1回だけ）

- [ ] Discord招待リンクを発行（無期限）
- [ ] 入会完了メールのテンプレートを決済システムに設定
- [ ] #自己紹介 チャンネルに案内文を固定投稿
- [ ] #操作方法・よくある質問 チャンネルにFAQを固定投稿

### 新規会員ごと

- [ ] 決済完了を確認
- [ ] 入会完了メールが送信されたことを確認
- [ ] （任意）Discord参加を確認、未参加なら1週間後にリマインド

---

## トラブルシューティング

| 問題 | 対応 |
|------|------|
| 招待リンクが無効になった | 新しいリンクを発行し、メールテンプレートを更新 |
| 会員がDiscordに参加できない | DMで個別サポート、リンクを再送 |
| 非会員が紛れ込んだ | DMで確認、会員でなければキック |
| 会員が退会した | Discordからも手動でキック |
