## ステップ2：ビジネスモデルチェック

> **目的**: サブスク型の儲かる仕組みを持っているか確認する

| # | チェック項目 | ✓ |
|---|-------------|---|
| 2-1 | サブスクリプション型（月額・年額）の商品・サービスを扱っているか？ | □ |

## ステップ3：業績チェック

> **目的**: 業績が成長しているか確認する（決算書を読む前に見るべきポイント）

### 決算書を見る前にチェック

| # | チェック項目 | ✓ |
|---|-------------|---|
| 3-1 | 社長のインタビュー動画を見て、ビジョンや熱意を感じるか？ | □ |
| 3-2 | 商品レビュー（Amazon、App Store等）で高評価が多いか？ | □ |
| 3-3 | 採用ページを見て、人を増やしているか（成長している証拠）？ | □ |

### 業績グラフをチェック

| # | チェック項目 | ✓ |
|---|-------------|---|
| 3-4 | 売上高のグラフは右肩上がりか？ | □ |
| 3-5 | 利益のグラフは右肩上がりか？（または「良い赤字」か？） | □ |
| 3-6 | 過去3年間で売上が成長しているか？ | □ |

### 「良い赤字」の見分け方
- **良い赤字**: 広告費や設備投資など、成長のための先行投資で赤字になっている
- **悪い赤字**: 商品が売れない、競争に負けているなどで赤字になっている

### 判定
- **3-4〜3-6のうち2つ以上に✓** → ステップ4へ進む
- **それ以外** → この銘柄は見送り（他の銘柄を探す）

## ステップ5：割安性チェック

> **目的**: 成長に対して株価が高すぎないか確認する

| # | チェック項目 | ✓ |
|---|-------------|---|
| 5-1 | PER（株価収益率）は10倍〜15倍の範囲か？ | □ |
| 5-2 | または、成長株なら15倍〜25倍の範囲か？ | □ |
| 5-3 | PERが10倍以下の場合、「訳あり銘柄」ではないか確認したか？ | □ |
| 5-4 | 同業他社と比べて、PERが極端に高くないか？ | □ |

### PERの目安

| PER | 判断 |
|-----|------|
| **10倍以下** | ⚠️ 要注意（訳あり銘柄の可能性あり） |
| **10〜15倍** | ✅ お買い得ゾーン |
| **15〜25倍** | ✅ 成長株なら許容範囲 |
| **25倍以上** | ⚠️ 割高の可能性あり（成長性を再確認） |

### 判定
- **5-1または5-2に✓、かつ5-4に✓** → **買い候補！**
- **それ以外** → この銘柄は見送り（他の銘柄を探す）
