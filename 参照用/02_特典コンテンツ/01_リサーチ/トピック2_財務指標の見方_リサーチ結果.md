# トピック2：財務指標の見方（PER・PBR・ROEで銘柄を評価する）_リサーチ結果

## 1. 一般的な説明方法・解説パターン

### パターンA：買い物・ショッピング比喩型
*   **PER（株価収益率）**: 「投資資金の回収期間」。
    *   解説: 「このお店を100万円で買ったら、毎年いくら利益が出る？ 毎年10万円利益が出るなら、10年で元が取れるね。これがPER10倍という意味」と説明。
*   **PBR（株価純資産倍率）**: 「解散した時の分け前」。
    *   解説: 「会社が今すぐ解散したら、株主にいくら戻るか。PBR1倍なら投資額がそのまま戻る、1倍以下ならお釣りが来る（バーゲンセール）」と説明。
*   **ROE（自己資本利益率）**: 「経営者の成績表」。
    *   解説: 「投資家から預かったお金を使って、どれだけ効率よく稼いだか。利回りみたいなもの」と説明。

### パターンB：図解・バランスシート連携型
*   企業の「お財布（B/S）」と「稼ぎ（P/L）」の図を描き、どこを比べているかを矢印で示す。
    *   PER = 株価（市場の評価） / 1株益（稼ぎ）
    *   PBR = 株価（市場の評価） / 1株純資産（お財布の中身）
*   視覚的に「分子」と「分母」の関係を示す方法。

### パターンC：フィルター（ふるい）型
*   投資候補を絞り込むための「ものさし」として紹介。
*   1. PERで割高なものを避ける。
*   2. PBRで割安なものを探す。
*   3. ROEで稼ぐ力があるか確認する。
*   という順序でチェックリスト化する説明。

## 2. ベストプラクティス（効果的な教え方）

*   **「PER 15倍の壁」を基準にする**:
    *   解説: まずは「15倍」という基準値を覚えさせ、それより高いか低いかで大まかな判断をさせる（日経平均の平均値などを根拠に）。
*   **「PER×成長率」のセット解説**:
    *   解説: 単にPERが高い＝割高ではなく、「高成長企業は将来の利益を先取りしているからPERが高くてもOKな場合がある」ことを教える（PEGレシオ的な考え方の簡易版）。
*   **PBR 1倍割れ＝「訳あり物件」の可能性**:
    *   解説: PBRが低いことだけを理由に買うと、万年割安株（バリュートラップ）に捕まるリスクを、「安いのには理由がある（ボロいアパートなど）」と例えて注意喚起する。
*   **ROE 8〜10%の「合格ライン」**:
    *   解説: 日本企業の目標ラインである8%や、欧米基準の10%を「優良企業の合格ライン」として提示し、これを超えている企業のみを対象にするシンプルなルール作り。

## 3. よくある質問・誤解

1.  **「PERが低い株を買えば絶対に儲かる？」**
    *   誤解: 低PER＝お買い得と思い込む。
    *   注意点: 業績悪化懸念で株価が下がってPERが低くなっている「将来の赤字予備軍」の可能性がある。
2.  **「PER 50倍とか100倍は危険？」**
    *   誤解: 高すぎるPERは暴落すると思い込む。
    *   注意点: Amazonなどの超成長期はPERが数百倍でも株価が上がり続けた。成長率との見合いであることを補足。
3.  **「PBR 1倍を割れていれば安心？」**
    *   誤解: 資産価値があるから下がらないと思い込む。
    *   注意点: 企業の資産内容（在庫や機械など）が本当に帳簿通りの価値で売れるとは限らない。
4.  **「ROEとROA、どっちを見ればいい？」**
    *   誤解: 指標が多すぎて混乱する。
    *   注意点: 株主視点ならまずはROE（自分のお金に対するリターン）だけでOKと割り切らせる。
5.  **「業種によって目安は違う？」**
    *   誤解: すべての株を一律15倍で判断しようとする。
    *   注意点: IT企業は高く、銀行や建設は低い傾向があるなど、同業他社との比較が重要であることを伝える。

## 4. 実例・ケーススタディ

*   **割安是正の例：PBR 1倍割れ対策企業**
    *   解説: 東証の要請により、PBR1倍割れの企業が自社株買いや増配を行い、株価が急上昇した事例（例：地銀や商社など）。「PBR1倍割れ」が投資のチャンスになり得る実例として。
*   **成長株の例：高PERでも買い**
    *   解説: 過去のファーストリテイリング（ユニクロ）など、PERが高く見えても、その後利益が追いついて株価がさらに上がった事例。「成長が割高感を凌駕する」パターン。
*   **効率経営の例：高ROE企業**
    *   解説: キーエンスのように、工場を持たない（ファブレス）などで極めて高い利益率とROEを叩き出し、株価が長期上昇している事例。
