# トピック3：売買タイミングの判断軸（テクニカル分析の基本）_リサーチ結果

## 1. 一般的な説明方法・解説パターン

### パターンA：信号機・天気予報型
*   **移動平均線**: 「トレンドの天気図」。
    *   上向きなら「晴れ（上昇）」、下向きなら「雨（下落）」。雨の日に傘を持たずに外出しない（下落トレンドで買わない）という教え方。
*   **RSI**: 「過熱感の温度計」。
    *   70度以上は「暑すぎ（買われすぎ）」、30度以下は「寒すぎ（売られすぎ）」。そろそろ反転するかも？というシグナルとして説明。

### パターンB：ゴールデンクロス・デッドクロス重視型
*   チャート上に2本の線（短期・長期）を引き、それがクロスする場所を「売買サイン」として機械的に教える方法。
*   「突き抜けたら買い」「割り込んだら売り」というビジュアル重視の解説。

### パターンC：逆張り・順張り使い分け型
*   相場には「トレンド相場（一方向に動く）」と「レンジ相場（一定の幅で動く）」があることを前提に説明。
*   トレンド相場では「移動平均線（順張り）」に従う。
*   レンジ相場では「RSI（逆張り）」に従う。
*   この使い分けこそが初心者の脱却ポイントと説く。

## 2. ベストプラクティス（効果的な教え方）

*   **RSIの「30/70」ルール**:
    *   解説: 複雑な計算式は省き、「30以下で買いゾーン、70以上で売りゾーン」というシンプルな数値基準を持たせる。ただし、「強いトレンドが出ている時は70を超え続ける（から売りではない）」という例外もセットで教えるのが重要。
*   **移動平均線の「向き」を見る**:
    *   解説: 交差（クロス）だけでなく、線の「傾き」を重視させる。線が上を向いている間はホールド、横ばいや下向きになったら警戒、という直感的な判断。
*   **「感情を排する」ためのルール作り**:
    *   解説: テクニカル分析を使う最大のメリットは「値ごろ感」や「恐怖・強欲」を排除し、サインに従って淡々と売買できることだと強調する。
*   **「頭と尻尾はくれてやれ」のマインドセット**:
    *   解説: RSIで底（0%）や天井（100%）を狙いすぎず、30%や70%といった「腹八分目」で動くことの重要性を説く。

## 3. よくある質問・誤解

1.  **「ゴールデンクロスが出たら絶対上がる？」**
    *   誤解: 100%当たる必勝法だと思っている。
    *   対策: 「だまし（サインが出ても逆に動くこと）」が頻繁にあることを伝え、損切りラインの設定がセットであることを教える。
2.  **「RSIが低いのに下がり続けるのはなぜ？」**
    *   誤解: 売られすぎなら反発するはずという思い込み。
    *   対策: 強い下落トレンド（ナイフが落ちている状態）ではRSIが機能しないことを説明し、移動平均線との併用を勧める。
3.  **「どの期間設定（日足、週足）を見ればいい？」**
    *   誤解: 短期売買をしないのに5分足を見て一喜一憂する。
    *   対策: 50-60代の投資スタイル（中長期）なら、「日足」や「週足」をメインに見るべきと指導し、ノイズを減らす。
4.  **「テクニカルだけで勝てる？」**
    *   誤解: 業績を見ずにチャートだけで勝負しようとする。
    *   対策: 銘柄選び（ファンダメンタルズ）で良い株を選び、売買タイミング（テクニカル）で安く買う、という「ハイブリッド」が最強であると説く。

## 4. 実例・ケーススタディ

*   **エントリーの成功例：押し目買い**
    *   解説: 上昇トレンド（移動平均線が上向き）の中で、一時的に株価が下がり、RSIが30付近まで落ちたタイミング。ここが「絶好の買い場（押し目）」であったチャート事例を示す。
*   **利確の成功例：過熱感での売り**
    *   解説: 株価が急騰し、RSIが80を超えたあたりで一部売却。その後株価が調整したチャートを見せ、「欲張らずに売っておいてよかった」事例として紹介。
*   **失敗回避例：デッドクロスでの逃げ**
    *   解説: 業績は悪くなさそうだが、チャートでデッドクロスが発生。ルール通り売却したら、その後悪いニュースが出て暴落。「チャートはニュースより早く動く」ことの証明事例。
