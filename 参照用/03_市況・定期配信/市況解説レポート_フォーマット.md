# 市況解説レポート：週2回配信フォーマット

## 全体コンセプト

| | **月曜配信** | **木曜配信** |
|---|---|---|
| **名前** | 📰 週刊マーケットブリーフィング | 🎓 相場の読み方トレーニング |
| **一言で** | 「今週、何を見ればいいか」 | 「どう考えればいいか」 |
| **素材** | 直近のニュース・経済指標 | 今週実際に起きた値動き |
| **提供する価値** | 相場の地図（視点と方向感） | 分析の型（再現可能な思考法） |
| **分量** | 約1,200〜1,500字 | 約1,500〜2,000字 |
| **つながり** | 木曜の伏線を張る | 月曜の内容を回収・深掘りする |

---

## 📰 月曜配信：週刊マーケットブリーフィング

> **読者の読後感：「今週はここを見ればいいんだ」**

| セクション | 内容 | 字数目安 |
|---|---|---|
| **① 今週の相場キーワード** | ニュース×影響を1行で要約。例：「米CPI鈍化 × 円安加速 → 輸出関連に追い風か」 | 1行 |
| **② 先週〜週末のニュース整理** | 重要ニュースを2〜3個に絞り、「なぜ株価に影響するのか」を各1〜2文で解説 | 300〜400字 |
| **③ 今週の注目セクター / テーマ** | マクロ環境から導かれる注目セクターを2〜3つ提示（※個別銘柄名は出さない） | 300〜400字 |
| **④ あい先生の視点** | 上記を踏まえた個人の相場観・スタンスをコメント | 200〜300字 |
| **⑤ 今週の観察ポイント** | 「このセクターで○○が起きたら注目」など、木曜レポートへの伏線を張る | 100〜200字 |

### 注意点
- 個別銘柄名は出さず、**セクター・テーマ単位** に留める
- 「上がる/下がる」の断言は避け、「○○なら追い風」という条件付きの表現にする

---

## 🎓 木曜配信：相場の読み方トレーニング

> **読者の読後感：「次から自分でもこう分析できそう」**

| セクション | 内容 | 字数目安 |
|---|---|---|
| **① 今週の学びテーマ** | 1行で学習テーマを提示。例：「決算発表後の"初動"をどう読むか？」 | 1行 |
| **② ケーススタディ** | 今週実際に起きた値動きを題材に解説。銘柄名は出すが、あくまで **過去の事実の振り返り** として扱う。「○○が起きた → 株価はこう動いた → なぜか？」の因果を分解 | 400〜500字 |
| **③ 判断の型（ここが本体）** | ケースから抽出できる **汎用的な思考法・チェックポイント** を言語化。読者が他の場面でも使える"型"に昇華させる | 300〜400字 |
| **④ セルフチェック問題** | 「来週○○セクターで似た局面が来そう。あなたならどう判断しますか？」 答えは出さず、考えるきっかけだけ渡す → 次の月曜で軽く振り返る | 200〜300字 |
| **⑤ あい先生のひとこと** | 自身の経験・失敗談を交えたコメント。人間味とファン化 | 100〜200字 |

### 注意点
- 銘柄は「過去にこう動いた」という **事実ベース** のみで扱い、将来予測はしない
- 提供するのは「答え」ではなく「考え方の型」

## 🔁 月曜 ↔ 木曜の連動サイクル

| **月曜** | **木曜** | **翌月曜** |
|---|---|---|
| 「今週は半導体セクターと決算ラッシュに注目」<br>「決算後の初動に注意して観察してみて」<br><br>↓ **伏線** | 「実際に○○の決算でこう動いた。」<br>「ここから学べる"型"は…」<br>「来週、○○で似た局面が来そう。どう判断する？」<br><br>↓ **回収＆次の伏線** | 「先週のセルフチェックの振り返り」<br>「今週の注目テーマは…」<br><br>↓ **継続** |

このサイクルが回ることで、読者は **毎週レポートを読む理由** ができ、継続率が上がる。

---

## ⚖️ 法的安全性チェックリスト

| 要素 | 対応 |
|---|---|
| 個別銘柄の将来予測 | ❌ しない |
| 個別銘柄の過去の値動き解説 | ✅ 事実として扱う |
| セクター・テーマの方向感 | ✅ 条件付きで提示 |
| 「買い/売り」の推奨 | ❌ しない |
| 思考法・判断基準の教育 | ✅ メインコンテンツ |

---

## 設計ポイント

1. **再現性のある「型」を教える** — 毎回同じフォーマットで配信することで、読者は自然と「マクロ → セクター → ケース → 思考法」という分析の型を身につける
2. **断言しない、でも逃げない** — 予測ではなく思考法を提供することで、投資助言に該当せず、かつ外れるリスクもない
3. **接続性を持たせる** — 月曜の観察ポイント → 木曜のケーススタディ → 翌月曜の振り返りで、2つのレポートが1セットとして機能する
4. **視覚的に読みやすく** — 表、箇条書き、絵文字アイコンを活用して、スマホでもスキャンしやすいレイアウトにする
