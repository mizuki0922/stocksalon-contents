# ポジショニングマップ

## 目的
競合と自社の立ち位置を可視化し、勝てる余白を明確にする。

## マップ1：専門性の深さ × 伴走・コミュニティ強度
**縦軸**：専門性の深さ（低 → 高）  
**横軸**：伴走・コミュニティ強度（弱 → 強）

```
        専門性：高
             |
             |   複眼経済塾（高/中）
             |          Seeking Alpha（高/弱）
             |   Investors Underground（高/強）
             |
             |
-------------+----------------------------- 伴走・コミュニティ：強
             |
             |   Bear Bull Traders（中/強）
             |   Market Hack Salon（中/中）
             |   投資家バーSTEP（低/強）
             |
             |   PostPrime（中/弱）
             |   杉原杏璃サロン（低/中）
             |
        専門性：低
```

**狙う位置（あい先生サロン）**  
「専門性：中〜高 × 伴走・コミュニティ：強」  
＝中高年向けにわかりやすく"判断軸"を渡し、安心して相談できる居場所。

---

## マップ2：学習/判断軸の明確さ × 参加ハードル
**縦軸**：学習/判断軸の明確さ（低 → 高）  
**横軸**：参加ハードル（低 → 高）

```
        学習/判断軸：高
             |
             |   ファイナンシャルアカデミー（高/高）
             |   複眼経済塾（高/中）
             |   株の学校ドットコム（高/中）
             |
-------------+----------------------------- 参加ハードル：高
             |
             |   Seeking Alpha（中/高）
             |   Warrior Trading（中/高）
             |   PostPrime（低/低）
             |   投資家バーSTEP（低/低）
             |
        学習/判断軸：低
```

**狙う位置（あい先生サロン）**  
「学習/判断軸：高 × 参加ハードル：低〜中」  
＝難語を避け、判断軸ツールで迷いを減らす"ゆっくり設計"。

---

## 補足
- 位置はリサーチ結果からの相対評価（目安）
- 今後の訴求・機能追加で位置が動く前提
