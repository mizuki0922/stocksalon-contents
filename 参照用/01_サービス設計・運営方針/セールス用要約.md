# セールス用要約（必須情報のみ）

## サロン名
**あい先生の株コンパスサロン**（略称：株コンパスサロン）

## 1. 一行定義
守りながら増やす個別株｜個別株の判断軸を身につける伴走サロン

## 2. 冒頭リード
銘柄選びと売買タイミングの迷いをなくす伴走サロン。

## 3. 提供内容の核
- 市況まとめ：週2回配信＋重大局面は臨時配信
- 伴走（質問→回答）：LINEでいつでも質問可能
- 月1回 Zoomオンライン勉強会
- 不定期 オフ会/交流イベント

## 4. 参加特典
- **銘柄選定チェックリスト（シート形式）**: 判断軸をチェックリスト化
- **メルマガ（3日に1回配信）**: 銘柄選びのノウハウを継続的に提供
- **判断軸コンテンツ3本（PDF）**: 銘柄選び・財務指標・売買タイミングの判断軸を体系的に解説
