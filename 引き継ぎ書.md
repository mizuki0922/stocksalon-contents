# オンラインサロン中身構築 引き継ぎ書

## 目的・この引き継ぎ書の使い方

- **対象**: サービス**購入後**のサロン運営・コンテンツ構築を行う担当者向け
- **ローンチ**（申込獲得・ステップ配信・VSL等）は本フォルダ（あい先生/オンラインサロン）内で行うため、本引き継ぎでは扱わない
- 本ドキュメントと「参照用」フォルダ内のファイルを基に、**会員が入会した後に提供するサービスの中身**を構築・運営する
- **タスク成果物**は「参照用/タスク成果物/」内の各フォルダに、1ファイルずつ追加していく形で進める

---

## サロン概要（購入後提供物の前提）

### サロン名・コンセプト
- **サロン名**: あい先生の株コンパスサロン（略称：株コンパスサロン）
- **コンセプト**: 判断軸を身につけ、守りながら増やす伴走型サロン
- **一行定義**: 守りながら増やす個別株｜個別株の判断軸を身につける伴走サロン

### 価格・形式
- **月額**: 9,980円（税込）
- **形式**: オンラインサロン（月額制・いつでも解約可能）

### サービス内容（購入後に提供するもの）
| 項目 | 内容 |
|------|------|
| 市況まとめ | 週2回配信（固定）。会員サイトにコンテンツ掲載＋Discordチャンネルにも投稿。重大局面は臨時配信で補完 |
| 相談導線 | Discord内でいつでも質問可能 |
| オンライン勉強会 | 月1回 Zoom。銘柄選定の方法を毎月体系的に公開。過去録画はオンデマンド提供（基礎テーマ1〜3は新規メンバー必須視聴） |
| オフ会・交流 | 不定期開催。コミュニティ強化 |

### 参加特典（入会時提供）
- **銘柄選定チェックリスト**（シート形式）: 判断軸をチェックリスト化
- **メルマガ**: 3日に1回配信。銘柄選びのノウハウを継続提供
- **判断軸コンテンツ3本（動画講義）**: ①銘柄選びの判断軸 ②財務指標の見方 ③売買タイミングの判断軸  
  ※ 動画講義のため、**資料作成・撮影・編集**が必須。

### ターゲット・運営トーン
- **メイン**: 50〜60代、会社員・主婦・個人事業主。「迷っている投資実践者」
- **悩み**: 銘柄選び・売買タイミングが分からない、情報過多で判断できない
- **運営トーン**: 専門用語を避け、ゆっくり噛み砕く。50〜60代に優しい設計。「答え」ではなく「判断軸」を提供し、自立を促す

### 設計・運営方針
- 最小構成で立ち上げ、60点で売り始める。会員が増えてから改善・作り込み
- 金商法リスク回避：銘柄推奨・投資判断の代行は行わず、判断軸の提供に留める

※ 詳細は「参照用/01_サービス設計・運営方針/」の各ファイルを参照（後ほど追加予定）。

## 体制・役割・前提の補足

| 項目 | 内容 |
|------|------|
| **サービス名** | 株コンパスサロン |
| **決済システム** | サブスクペイ |
| **運営開始日の定義** | 初回決済が入った日 |
| **担当体制** | 1名 |
| **銘柄分析AIツール（09）** | 現在開発中。会員サイト（stock-research-tool）内の一機能として提供予定 |

### 会員サイト・配信の前提
- **会員サイト**: 株コンパスサロンの会員向け機能は **stock-research-tool** 内で提供（決済・教材・管理画面を一括構築）。オンクラスは使用しない
- **コミュニティ**: BANDは使用せず **Discord**（Botによるロール付与・剥奪）
- **公式LINE**: 告知用。会員サイト更新時・市況まとめ・勉強会などの告知も行う
- **市況まとめ**: 会員サイトにコンテンツとして掲載し、Discordの該当チャンネルにも投稿する

### コンテンツ制作の役割分担
- **コンテンツ制作者**: 市況まとめ・勉強会資料など、専門知識が必要な原稿・コンテンツの**作成**を担当
- **講座責任者**: コンテンツ制作者への**ディレクション**（指示・確認・承認フロー）を担当
- **あい先生（iSense）**: 講座責任者ではなく**演者**（勉強会の登壇・出演など）

※ 参照用フォルダ（01_サービス設計・運営方針、02_特典コンテンツ 等）の中身は後ほど追加予定。

- **サポート体制・対応フロー**: 24時間〜2営業日返信、返信漏れ防止、品質基準、返信なし時の対応 → `15_サポート・返金・臨時配信/`
